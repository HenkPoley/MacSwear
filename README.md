# MacSwear

MacSwear is a native port of Swear for macOS.

A kind of 'Nokia Snake' but in 3D. _Tip:_ Game settings are in the menus at the top of your screen.

The game is released in the Public Domain by Dag Ågren/WAHa.06x36.

Ported to macOS 14 Sonoma using OpenAI Codex with GPT-5.5. Still uses the deprecated OpenGL, which still works.
